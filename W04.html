This assignment introduces simplified scripting with jQuery.

The code is available at https://bitbucket.org/professorcase/w04.

Results of the unit tests are available at test/test.html.


Currently, the application gets the user's name and the size of their ranch. The app calculates square miles and estimates how many sheep they can hold on their land.


Your goal is to create a new app using jQuery that calculates the number of seats reserved at the next Bearcats sporting event.


Your new application should:

  1. Ask the name of the ticket buyer.
  2. Ask the organization for which they are buying tickets.
  3. Ask the number of rows the organization wants to reserve.
  4. Ask the number of seats per row the organization wants to reserve.
  5. Calculate the number of seats requested (rows * seats per row). Create and use a calculateSeats() function that returns the number of seats given number of rows and seats per row. If the number cannot be calculated or inputs are bad or too large, it should return 0.
  6. Display the number of students that will have a seat (assume 1 student per seat).
  7. Display one Bearcat logo (or other image of your choice) for each seat.
  8. Display a unique application name in the nav bar.
  9. Use variable names and prompts appropriate to your new application.

Click the button to start the app.
